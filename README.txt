--------------------------------------------------------------------------------
                           Instruções de Compilação
--------------------------------------------------------------------------------

- Para compilar, execute o arquivo compile.bat (Windows) ou compile.sh (Linux).

--------------------------------------------------------------------------------
                            Instruções de Execução
--------------------------------------------------------------------------------

- Para executar, abra o arquivo Trabalho.exe.
- É recomendado utilizar o terminal com largura de, no mínimo, 235 caracteres.
  Para isso, pode-se diminuir a fonte do terminal, e utilizar no modo
  maximizado.
